## AgentPage优化，prompt工程
1. 引入 Chain-of-Thought (COT) 思维链
你让 AI 在优化简历时，先分析优缺点，再给出改进建议，最后输出优化后的内容，而不是直接输出最终答案。
这样可以让 AI 的输出更有条理、推理更充分，提升了内容质量。
2. 加入 Few-shot 示例
对于“自我评价优化”，你在 prompt 里给了两个高质量的优化示例（原文、分析、改进后），让 AI 模仿这种风格和深度。
这样能让 AI 输出的内容更贴近你想要的格式和表达方式。
3. 针对不同优化类型定制 prompt
针对“自我评价”、“技能”、“项目/经历”等不同 quick action，分别设计了不同的 prompt 模板。
每个模板都要求 AI 先分析、再建议、再输出优化内容，部分还给出了示例。
4. 面试模式也引入了分步反馈
在面试模式下，prompt 要求 AI 逐条提问，并在每次回答后给出反馈和改进建议，而不是一次性输出所有内容。
5. 默认 prompt 也要求分步分析和建议
即使没有命中特定 quick action，默认 prompt 也要求 AI “step by step” 分析优缺点，并给出具体改进建议。
